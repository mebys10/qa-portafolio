# Test Cases

## TC-001 - Valid Login

### Steps
1. Open login page
2. Enter valid email
3. Enter valid password
4. Click Login button

### Expected Result
User logs in successfully.


---

## TC-002 - Invalid Login

### Steps
1. Open login page
2. Enter invalid password
3. Click Login button

### Expected Result
Error message should appear.
